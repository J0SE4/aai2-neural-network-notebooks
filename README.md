# AAI2 — Notebooks de redes neuronales

Notebooks de prácticas de AAI2 con ejercicios de feed-forward, backpropagation, CNN e hiperparámetros.

## Tecnologías
- Python 3
- Jupyter Notebook
- PyTorch / Torchvision
- NumPy, Matplotlib, tqdm, Optuna

## Preparación
```bash
python -m venv .venv
# Windows:
. .venv/Scripts/activate
pip install -r requirements.txt
```

## Ejecución
```bash
jupyter notebook
```
